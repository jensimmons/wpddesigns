The Technology that Runs the Web.

The web is made of code — HTML, CSS, Javascript and more. Web Platform is an effort document web platform technologies in a place everyone to use. The W3C, Adobe, Facebook, Google, HP, Intel, Microsoft, Mozillia, Nokia, Opera and others have gathered together to create a central resource for looking up the latest, official information on the code web developers use everyday.
